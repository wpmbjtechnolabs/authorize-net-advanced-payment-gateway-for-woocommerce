Authorize.net Advanced Payment Gateway For WooCommerce extends the functionality of WooCommerce to accept payments from credit/debit cards.
== Description ==
= Introduction =

Easily add Authorize.net options to your WordPress / WooCommerce website.
 
Why Authorize.net Advanced Integration Method (AIM) is right for your store:

* Free Authorize.net Support 
* Accept international transactions from customers worldwide 
* Free Fraud-Prevention tools and Maximum Security 
* Customer checkout experience is highly customizable 
* Customers never leave your site during checkout
* Accepts all Major Credit Cards / Debit Cards – Visa®, MasterCard®, American Express®, Discover®, Diner’s Club, JCB

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Authorize.net Advanced Payment Gateway For WooCommerce" and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting previous versions if they exist
2. Activate the plugin in your WordPress admin area.


= configuration =

Easy steps to install the plugin:

*	Upload authorize-net-advanced-payment-gateway-for-woocommerce folder/directory to the /wp-content/plugins/ directory
*	Activate the plugin through the 'Plugins' menu in WordPress.
*	Go to WooCommerce => Settings
*	On the "Settings" page, select "Checkout" tab.
*	Under "Payment Gateways" you will find all the available gateways, select "Authorize.net" option
*	On this page you wil find option to configure the plugin for use with WooCommerce
*	Enter the API details (Login ID, Transaction Key) [obtaining username and password](http://reseller.authorize.net/application/?resellerId=27457).

Configurable elements:
Title: This controls the title which the user sees during checkout.

*	Description: This controls the description which the user sees during checkout.

*	Login ID: This is the API Login provided by Authorize.net when you signed up for an account.[Login ID](http://reseller.authorize.net/application/?resellerId=27457).

*	Transaction Key: This is the Transaction Key provided by Authorize.net when you signed up for an account.[Transaction Key](http://reseller.authorize.net/application/?resellerId=27457).

*	Transaction Success Message: This message will appear upon successful transaction. You can customize this message as per your need.

*	Transaction Failed Message: This message will appear when transaction will get failed/declined at payment gateway.

*	API Mode: Sendbox(testing mode) / live (production mode)
